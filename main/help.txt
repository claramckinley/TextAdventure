DIRECTIONS:
	you can use use north, south, east, west, northwest, southwest, sortheast, southest, up, or down to navigate the town
	
COMMANDS:
	type EXAMINE to look at the space you are in
	type PICK UP/TAKE to pick up and object
	type DROP to drop an object
	type SHOOT when you hear the monster (if you have a gun and a bullet)
	type DIG to unearth something
	type INVENTORY to see what youre holding
		
	type TIME to see how long you have played
	type MOVES to see how many moves you have taken
	type SCORE to see your score
	type SANITY to see how much sanity you have left
	type QUIT to end the game
